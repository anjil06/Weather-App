# 🌦️ Weather App

A simple and responsive Weather Application that allows users to check real-time weather conditions of any city using a weather API.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Displays temperature in real-time
* ☁️ Shows weather conditions (Cloudy, Rainy, Sunny, etc.)
* 💧 Humidity information
* 🌬️ Wind speed details
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Weather API (e.g., OpenWeather API)

## ⚙️ How It Works

1. User enters a city name
2. App sends request to Weather API
3. API returns weather data
4. Data is displayed on the screen

---

## 📦 Installation & Setup

1. Clone the repository

```
git clone https://github.com/anjil06/Weather-App.git
```

2. Navigate to the project folder

```
cd Weather-App
```

3. Open `index.html` in your browser

---

## 🔑 API Setup

1. Get a free API key from OpenWeather
2. Replace the API key in your JavaScript file:

```javascript
const apiKey = "YOUR_API_KEY";
```
